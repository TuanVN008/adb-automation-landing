4) Curriculum (list bài giảng tổng quan)

Module 0 — Khởi động

0.1 Tổng quan ADB Automation & mô hình giả lập

0.2 Cài đặt ADB, PATH, kiểm tra thiết bị adb devices

0.3 Kết nối LDPlayer/BlueStacks/Boxphone, mapping cổng ADB

Module 1 — Nền tảng Lệnh & Hàm Cốt lõi

1.1 tap(x,y): Click chuẩn xác (tọa độ, density, rotation)

1.2 swipe(...): Vuốt, kéo, scroll dài (tối ưu duration)

1.3 type(text): Nhập text, dán clipboard, ký tự đặc biệt

1.4 back, home, openApp(package), killApp(package)

1.5 waitFor(condition, timeout), sleep(ms), retry(n)

Module 2 — Nhận diện hình ảnh & Click theo ảnh

2.1 Khái niệm template matching, ngưỡng (threshold)

2.2 Chụp màn, tìm ảnh vùng (ROI), đa độ phân giải

2.3 findImage(template).click() | fallback to coordinates

2.4 Chống sai số: nhiều template, nhiều trạng thái UI

2.5 Benchmark & tối ưu tốc độ nhận diện

Module 3 — Vòng lặp & Kịch bản

3.1 Cấu trúc loop an toàn (break/continue/guard)

3.2 Xử lý ngoại lệ, re-attach khi giả lập lag/disconnect

3.3 Kịch bản đa bước: mở app → login → tác vụ → log out

3.4 Lập lịch tác vụ (cron-like) & queue đơn giản

Module 4 — Đa phiên & Điều phối

4.1 Quản lý nhiều instance (ADB port, device id)

4.2 Pool phiên + throttle để tránh nghẽn CPU/GPU

4.3 Logging tập trung & theo dõi trạng thái từng phiên

4.4 Khởi tạo/đóng batch instances theo cấu hình JSON

Module 5 — Ổn định & Chống lỗi

5.1 Timeout chuẩn, retry có backoff

5.2 Phục hồi khi app crash / UI treo

5.3 Lưu checkpoint (bước hiện tại) — resume mạnh mẽ

5.4 Clean-up: clear cache, force-stop, reboot instance

Module 6 — Tooling & CI

6.1 Cấu trúc dự án base (thư mục, tách layer, config .env)

6.2 CLI điều khiển (start/stop/status)

6.3 Ghi log JSONL, export CSV, gửi báo cáo Telegram/Discord

6.4 Test nhanh kịch bản, mock UI, screenshot diff

Module 7 — Case thực chiến

7.1 Auto đăng nhập + nhập OTP/sms

7.2 Auto tìm & bấm nút theo ảnh (đa trạng thái)

7.3 Auto farm thao tác lặp (ví dụ nhiệm vụ hằng ngày)

7.4 Tinh chỉnh hiệu năng & scaling đa máy

Module 8 — Mở rộng & Bảo trì

8.1 cập nhật base khi ADB/giả lập thay đổi

8.2 chống “UI drift”: thêm/bớt template linh hoạt

8.3 checklist phát hành phiên bản bot mới
